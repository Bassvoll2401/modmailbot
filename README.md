# Discord Modmail
A Discord Modmail Bot Base On Modmail.js 

#### Simple Setup/Startup

1. Edit [`.env`](./.env)
2. Replace `TOKEN`, `PREFIX` And `INBOX`
3. Run [`start.bat`](./start.bat)

#### Require Packages

- Discord.js
- Discord-Sender
- Dotenv 

#### Any Problem ?

Check Out [modmail.js Homepage](https://github.com/modmail.js) To Learn More
